## 4. 設計含義

| 觀察 | 對應計畫書設計 |
|------|----------------|
| Q/K 為準標準常態（mean ≈ 0, std ≈ 1） | 動態稀疏假設成立：少數離群值主導 |
| 動態範圍 ±6（≈ 6σ） | INT8 可完整表達；INT4 邊緣（±7） |
| 離群值稀有（>3σ 約 0.3%） | 雙路徑設計合理：少數 token 走 8-bit、多數走 4-bit |
| 4-bit 截斷誤差不可忽略 | 二階多項式 Softmax 補償有意義 |

## 5. 一句話結論

> **Q/K 數值分佈符合計畫書動態稀疏假設，雙路徑設計（4-bit / 8-bit）有數據基礎。**
